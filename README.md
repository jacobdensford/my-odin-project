# My Odin Project

*My notes and progress on [The Odin Project](https://www.theodinproject.com/).*

From January 2025 to June 2026 I learned full stack web development by working my way through The Odin Project (referred to as TOP going forward). It was a long and challenging journey. 

View my [timeline and progress](/notes/top-timeline.md). (Note that my initial completion estimates proved to be *way* more ambitious than my schedule ended up allowing for.)
